# 🧠 5-Whats Analyzer 技能包

這是一個專門用於深度問題分析的諮詢技能。它將 AI 轉化為一位風格犀利、冷靜的資深諮詢師，協助您穿透情緒表象，直擊問題核心。

## 🌟 核心特色
- **犀利提問**: 嚴格執行 5 Whats 邏輯，禁絕 Why 提問，更注重事實與邊界。
- **心理學背景**: 整合多項心理與管理理論，提供具深度的系統性分析。
- **自動化報告**: 完成對談後自動產出具備行動建議的正式報告。

## 🚀 如何使用？
1. 對 AI 執行指令：`@5-whats-analyzer` 或呼叫 `/5-whats-analyzer`。
2. 根據諮詢師的引導，如實回答每個階段的 What 提問。
3. 取得最終的分析報告。

## ⚙️ 模型規範
- 提問階段：Trigger -> Gap -> Boundary -> Pattern -> Pivot。
- 報告格式：包含對話紀錄、根源分析與行動策略。
