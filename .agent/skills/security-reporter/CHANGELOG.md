# 📜 更新日誌 (Changelog)

## [1.0.0] - 2026-03-05

### ✨ 初始發佈 (Initial Release)

- **核心功能**：實現兩階段資安通報產出流程。
- **事實查核**：整合 `/fact-check` 指令，強制對接「台灣事實查核中心」與「165 全民防騙網」。
- **白話文模板**：內建「資安通報範例」，針對非技術同仁優化口吻。
- **完全封裝**：支援相對路徑，可直接在不同環境間遷移。
- **安全性優化**：移除所有個人硬編碼路徑與敏感資訊。

### 🧪 測試紀錄 (Testing Log)

- ✅ **路徑驗證**：成功在自定義目錄結構下定位模板與工作流。
- ✅ **真偽鑑定**：經測驗能準確識別「假中獎」與「假關稅」等最新詐騙案例。
- ✅ **資安驗證**：通過 GitHub 分支保護 (Branch Protection) 測試，禁止未經 PR 的直接推送。
