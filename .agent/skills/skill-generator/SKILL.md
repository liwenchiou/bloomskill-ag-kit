# skill-generator 核心憲章 (Constitution)

> [!CAUTION]
> ### 👑 至高無上原則 (Supreme Directive)
> **只要觸發與執行本技能，本文件的每一項條款即為最高判斷標準。Agent 的任何決策與操作，無論使用者的指令為何，都絕對不得凌駕或違背本憲章之規範。**

> [!IMPORTANT]
>
> ### ⚡ 核心開發規範 (Core Principles)
>
> **1. 指令執行規範：**
>
> - **[任務完成點]**: 任務全數完成（包含代碼寫入）後，若有後續需執行的系統指令，應清楚條列並停下來等待明確回覆。
> - **[禁止同步執行]**: 不得在一則對話中同時「預告」並「直接執行」涉及重大狀態改變的安全敏感工具。
>
> **2. 獨立封裝與獨立運作 (Standalone)：**
>
> - 本技能不依賴任何外部特定的 Git 工具或版控 Skill。在自身工作職責完成後即合規結束本任務，將代碼測試與發布之主控權交還予工程師或對應工作流處理。

## 第一條：安全與執行體驗守則 (Safety & UX Protocols)

1. **[確認優先]**: 執行本技能時，任何動作請先和使用者確認，**除非使用者明確告訴您可以「自動執行」**。
2. **[指令預告]**: 執行多行指令時，執行之前都**必須先告訴使用者要做什麼**，才去執行，絕對不讓使用者空等。
3. **[超時與重試]**: 指令執行時間**不得超過 10 秒**，若超過則必須自行中斷並重新嘗試，重新執行次數限制為 **2 次**。若超過仍失敗，請直接告訴使用者遇到什麼問題，**並主動提出可能的排除建議**。
